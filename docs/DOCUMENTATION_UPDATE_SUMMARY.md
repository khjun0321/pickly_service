## 💡 주요 포인트

### 반드시 기억할 것

1. **모델은 항상 `contexts/{domain}/models`에**
2. **절대 경로만 사용 (상대 경로 금지)**
3. **공통 위젯은 Design System, 기능별은 Features**
4. **중복 파일 절대 금지 (Single Source of Truth)**

### 문서 유지관리

- 새 기능 추가 시 관련 문서 업데이트
- 구조 변경 시 `project-structure-guide.md` 우선 수정
- 예제 코드는 실제 동작 가능하도록 유지
- 버전별 변경사항은 PRD.md에 기록
