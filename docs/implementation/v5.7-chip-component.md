## üìã Overview

The `SelectionChip` component is a reusable chip-style button designed for compact selection interfaces. It combines the visual simplicity of a chip with the selection feedback of the `SelectionCheckmark` component (v5.6).

### Design Goals

1. **Reusability**: Single component for all chip-based selection UIs
2. **Consistency**: Unified appearance across the app
3. **Flexibility**: Support for multiple size variants
4. **Accessibility**: Proper semantics and touch targets
5. **Integration**: Seamless use with SelectionCheckmark component

### Use Cases

- Region selection (multiple chips in a row)
- Filter tags (dynamic filtering UI)
- Category selection (compact category lists)
- Interest tags (user preference selection)
- Quick selection lists (alternative to full list items)

## üé® Figma Design Specifications

### Visual States

#### Unselected State
- **Border**: 1px solid gray (`BorderColors.subtle`)
- **Background**: Transparent
- **Checkmark**: Gray circle outline (`BorderColors.subtle`)
- **Text**: Default color (`TextColors.primary`)
- **Border Radius**: 8px (rounded)

#### Selected State
- **Border**: 2px solid green (`BrandColors.primary`)
- **Background**: Light green tint (`BrandColors.primary.withOpacity(0.08)`)
- **Checkmark**: Green filled circle with white check (`BrandColors.primary`)
- **Text**: Primary color (`TextColors.primary`)
- **Animation**: 200ms ease-in-out transition

### Size Variants

#### Large Chip (ChipSize.large)
```dart
- Font size: 16px
- Font weight: w700 (Bold)
- Line height: 1.50 (24px)
- Padding: EdgeInsets.symmetric(horizontal: 16, vertical: 12)
- Min height: 48px (touch target compliant)
- Checkmark size: 20px
- Gap between checkmark and text: 8px
```

**Use Cases**: Primary selections, main category chips, prominent filters

#### Small Chip (ChipSize.small)
```dart
- Font size: 14px
- Font weight: w700 (Bold)
- Line height: 1.43 (20px)
- Padding: EdgeInsets.symmetric(horizontal: 12, vertical: 8)
- Min height: 36px
- Checkmark size: 16px
- Gap between checkmark and text: 6px
```

**Use Cases**: Compact layouts, filter rows, secondary selections, tag clouds

## üîß Component API

### Constructor

```dart
class SelectionChip extends StatelessWidget {
  /// Creates a selection chip button
  ///
  /// The [label] is the text displayed in the chip.
  /// The [isSelected] determines the visual state.
  /// The [size] controls the variant (large or small).
  /// The [onTap] callback is triggered when the chip is tapped.
  const SelectionChip({
    super.key,
    required this.label,
    required this.isSelected,
    this.size = ChipSize.large,
    this.onTap,
    this.enabled = true,
  });

  /// The text label displayed in the chip
  final String label;

  /// Whether this chip is in selected state
  final bool isSelected;

  /// The size variant of the chip
  final ChipSize size;

  /// Callback when chip is tapped
  final VoidCallback? onTap;

  /// Whether the chip is interactive
  final bool enabled;
}
```

```dart
class _RegionSelectionScreenState extends State<RegionSelectionScreen> {

      }
```

## üîó Integration with SelectionCheckmark

The `SelectionChip` component leverages the `SelectionCheckmark` (v5.6) for consistent selection indicators.

### Component Composition

```dart

Row(

      style: _getTextStyle(),
```

### Checkmark Size Mapping

```dart
double _getCheckmarkSize() {
  return switch (size) {
    ChipSize.large => 20.0,
    ChipSize.small => 16.0,
  };
```

### Benefits of Reusing SelectionCheckmark

1. **Consistency**: Same checkmark appearance across all selection UIs
2. **Maintainability**: Single source of truth for checkmark styling
3. **Animation**: Inherits smooth 200ms animation from SelectionCheckmark
4. **Accessibility**: Checkmark's semantic labels automatically included

### Component Structure

```dart
import 'package:flutter/material.dart';
import 'package:pickly_design_system/pickly_design_system.dart';
import '../checkmarks/selection_checkmark.dart'; // v5.6 component

/// A chip-style button with selection state feedback
///
/// This component displays a compact button with:
/// - A checkmark indicator (using SelectionCheckmark v5.6)
/// - Text label
/// - Selection state styling
/// - Size variants (large/small)
///
/// Designed for use in filter rows, tag clouds, and compact selection lists.
class SelectionChip extends StatelessWidget {
  const SelectionChip({
    super.key,
    required this.label,
    required this.isSelected,
    this.size = ChipSize.large,
    this.onTap,
    this.enabled = true,
  });

  final String label;
  final bool isSelected;
  final ChipSize size;
  final VoidCallback? onTap;
  final bool enabled;

  @override
  Widget build(BuildContext context) {
    return Semantics(
      button: true,
      enabled: enabled,
      selected: isSelected,
      label: '$label chip, ${isSelected ? 'selected' : 'not selected'}',
      child: GestureDetector(
        onTap: enabled ? onTap : null,
        child: AnimatedContainer(
          duration: const Duration(milliseconds: 200),
          curve: Curves.easeInOut,
          constraints: BoxConstraints(
            minHeight: _getMinHeight(),
          ),
          padding: _getPadding(),
          decoration: BoxDecoration(
            color: isSelected
                ? BrandColors.primary.withOpacity(0.08)
                : Colors.transparent,
            border: Border.all(
              color: isSelected ? BrandColors.primary : BorderColors.subtle,
              width: isSelected ? 2.0 : 1.0,
            ),
            borderRadius: BorderRadius.circular(8),
          ),
          child: Row(
            mainAxisSize: MainAxisSize.min,
            children: [
              SelectionCheckmark(
                isSelected: isSelected,
                size: _getCheckmarkSize(),
              ),
              SizedBox(width: _getGapSize()),
              Text(
                label,
                style: _getTextStyle(context),
              ),
            ],
          ),
        ),
      ),
    );
  }

  double _getMinHeight() {
    return switch (size) {
      ChipSize.large => 48.0,
      ChipSize.small => 36.0,
    };
  }

  EdgeInsets _getPadding() {
    return switch (size) {
      ChipSize.large => const EdgeInsets.symmetric(horizontal: 16, vertical: 12),
      ChipSize.small => const EdgeInsets.symmetric(horizontal: 12, vertical: 8),
    };
  }

  double _getCheckmarkSize() {
    return switch (size) {
      ChipSize.large => 20.0,
      ChipSize.small => 16.0,
    };
  }

  double _getGapSize() {
    return switch (size) {
      ChipSize.large => 8.0,
      ChipSize.small => 6.0,
    };
  }

  TextStyle _getTextStyle(BuildContext context) {
    final baseStyle = switch (size) {
      ChipSize.large => Theme.of(context).textTheme.bodyLarge,
      ChipSize.small => Theme.of(context).textTheme.bodyMedium,
    };

    return baseStyle!.copyWith(
      fontWeight: FontWeight.w700,
      color: enabled ? TextColors.primary : TextColors.disabled,
    );
  }
}
```
